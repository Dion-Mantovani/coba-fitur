# Halo Dunia! Ini Blog Hardcore Pertama Gw.

Menulis blog langsung dari kode itu rasanya puas banget. Kodenya enteng, performanya kencang, dan gak perlu pusing mikirin server database down.

## Keuntungan nulis blog via Markdown lokal:

- **Ngebut:** Langsung di-render jadi HTML statis saat build.
- **Aman:** File tulisan ikut kesimpan di GitHub lo.
- **Simpel:** Cukup ketik-ketik biasa pakai teks.
